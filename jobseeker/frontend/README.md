# Frontend

This directory contains the front-end files for the project.

## Directory Structure

- `static/css`: This directory is used to store CSS files for the front-end.
- `static/js`: This directory is used to store JavaScript files for the front-end.
- `static/images`: This directory is used to store image files for the front-end.
- `templates/index.html`: This file is an HTML template for the front-end. It can be used as the main page of the application.
- `app.py`: This file is the main Flask application file for the front-end. It contains the routes and logic for handling HTTP requests and rendering templates.

## Setup

To run the front-end application, follow these steps:

1. Install the required dependencies listed in `requirements.txt`.
2. Run the Flask application using the command `python app.py`.
3. Access the application in your web browser at `http://localhost:5000`.
